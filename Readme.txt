Provides Custom links at each page from available menus

== Description ==
An alternate to provide page links, external links for any page.
Provides shortcode to generate different links at any page.
Different Menus can be created and called on pages as per requirement.

Salient Features:

1. Customized links at each page
2. easily manage your links through menu



== Installation ==
1. Upload menurocker.php to the wp-content/plugins directory.
2. Activate the plugin through the Plugins menu in WordPress.
3. Select custom menu for each page in Edit/Add page.
